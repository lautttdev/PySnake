# PySnake

A recreation of the classic arcade game Snake built using Python and Pygame.

This project is part of a collection of game recreation made by me in order to learn about game development, game programming and game design. 
Every mechanic was implemented from scratch.

## Features
- Grid-based and tile-based movement.
- sprite management using spritesheets and arrays.
- Matrix-based map generation.
- Collision prediction before movement.

## What I learned and Improved
- Object-Oriented Programming (OOP).
- Working with spritesheets and arrays for sprite rendering.
- Game loops and state management.
- Collision detection and game logic design.

## Technologies
- Python 3.
- Pygame 2.

## Controls
- WASD or Arrow keys to move.

```
git clone https://github.com/lautttdev/PySnake.git
cd PythonInvaders
pip install pygame
python main.py
```
